Prerequisites:

    * Virtualenv
    * SQLite3

Installation
============

Run the following command:

    pip install -r requirements.txt

### Usage

Run the following command to use it locally:

    python manage.py migrate
    python manage.py runserver 0.0.0.0:8000

Done.
